# Mnsx-AchillesLink

> 轻量级、高性能 C++11 网络库。基于 Reactor 模式，支持非阻塞 TCP 服务端与客户端，采用事件驱动内核

## 项目简介

**Mnsx-AchillesLink**是工业级高性能异步网络库，作为后续项目的底层驱动引擎

它基于非阻塞I/O和事件驱动机制，旨在为上层业务提供极速、稳定且低延迟的字节流传输通路，是支撑万级并发连接的坚实地基

## 核心特性

* **高效的Reactor事件模型**
  * 基于Linux原生Epoll实现水平触发事件循环
  * 采用经典的One Loop Per Thread设计，完美支撑高频I/O事件的快速分发和处理
* **极致的零拷贝ByteBuffer**
  * 设计预留机制，缓冲区头部预留16字节空间，支持在不移动Body数据的前提下直接**零拷贝**封装协议头
  * 动态扩容和平移，内置自动**扩容策略**与数据**平移机制**，通过**双指针**高效管理字节流，彻底解决网络粘包和半包问题
* **Codec解耦架构**
  * 编解码器与网络传输逻辑完全剥离
  * Codec仅负责数据传输协议的静态转换，极大提升了协议的可测试性和复用性
* **合理的资源管理**
  * 大量使用C++11智能指针与RAII机制严格管理文件描述符与连接状态
  * 内置优雅关闭流程，确保在异常断开或系统退出时，所有底层资源安全回收
